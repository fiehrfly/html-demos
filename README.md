# My HTML Demos / Client Deliverables

This repo stores HTML files generated in Claude (e.g., client demos, proposals, or one‑off pages).

## How to use

- Each file is self‑contained (no server needed).
- Open in a browser by:
  - Clicking the file on GitHub → “Raw” → then opening in browser, or
  - Downloading and double‑clicking the `.html` file.

## Files

- `proposal-v1.html` – Client A proposal draft
- `client-x-demo.html` – Interactive demo for Client X
